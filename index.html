    3.What is an event loop and call stack

    Event Loop & call stack: A call stack is mechanism for an intrepreter to keep track of its place in ascript to call multiple functions-what function is currently running and what to be called within that function.
    The Event loop is constantly running process that monitors both are callback queue and call stack.All the web apis functions like setTimeout ,settimeInterval ,fetch calls etc..will not be directly executed inside the execution context,Firstly,they will be moved to the callback queue and then the moment call stack gets empty these functions will be pushed to call stck from call back queue by event loop.If the call stack is not empty event loop waits until it is empty and places the next function from callback queue to the callstack.

    4.What are promises and why do we need them?

    Promises are used to handle the asynchrous operations in Javascipt.They are easy to manage when dealing with the multiple asynchronous operations compare to callback hells and events.Multiple callback functions would create callback hell that leads to unmanageble code and it is also not easy for any user to handle multiple callbacks at the same time.So, promises are the ideal choice for handling asynchronous operations in simplest manner.A promise is in one of these states.They are :

  • Pending State:Intial state,neither fulfilled nor rejected.
  • fulfilled:The operation is fulfilled
  • Rejected:The operation is failed.
    Benifits Of Promises:
  • Code Readability.
  • Better handling of asynchrous operations
  • Better flow of control in asyncronous logic.

    •     var promise = new Promise(function(resolve,reject){
        const x="promise1";

        if(x===y){
            resolve();

        }
    })

    5.What is promise chaining

    the process of excuting the scequence of asynchronous tasks one after another using promises

    new Promise(function(resolve,reject){ setTimeout(()=>{resolve(1)},1000); }).then(function(result){ console.log(result); return result * 2; }).then(function(result){ console.log(result); return result * 4; }) .then(function(result){ console.log(result); return result * 6; })
